一、物的概念与种类（★★）

物权法上的物须满足：有体性、可支配性、在人体之外。行为、智力成果不是物权客体。

📌 物的种类速记

分类标准类型典型举例
流通性流通物/限制流通物(文物、黄金)/禁止流通物(专属国家)禁止流通：矿藏、水流等
移动性不动产(土地、房屋) vs 动产重要：不动产登记，动产交付
可替代性可替代物(粮食)/不可替代物(齐白石画)仅限于动产
可分性可分物(米、酒) / 不可分物(手机、牛)共有分割时意义
主从物从物辅助主物(钥匙、备用胎)无相反约定，从物随主物
原物孳息天然孳息(果实、幼畜) / 法定孳息(租金、利息)用益物权人优先取得天然孳息
✅ 真题点睛 (2021)：消费物与非消费物的分类仅限于动产；汽车属于不可分物；文物属限制流通物。

二、物权概念·种类·基本原则（★★★）

物权=直接支配+排他（所有权、用益物权、担保物权）。与债权区别：支配权vs请求权；绝对权vs相对权；排他性vs兼容性。

基本原则

  • 物权法定：种类、内容由法律规定。违反物权法定，物权创设无效，但不影响合同效力（叔侄“经营权”案 → 物权无效，但合同有效，叔叔无权要求返还店铺）。
  • 物权客体特定（一物一权）：所有权只能存在于一物之上，但所有权人可为多人。
  • 公示原则：动产交付/不动产登记。例外（土地承包经营权、地役权合同生效设立，登记对抗）。
📖 【案例·物权法定】叔侄约定创设“经营权”

侄子支付30万元，在叔叔店铺上享有排他性占有、使用、收益，期限20年，叔叔若与第三人订立合同转让，侄子可主张合同无效。
分析： 物权法定原则，“经营权”非《民法典》规定的用益物权种类 → 违反种类法定，物权行为无效；但债权合同仍有效，叔叔无权直接请求返还店铺（见2023真题）。

三、物权变动（★★★）

基于法律行为：动产原则交付生效，不动产原则登记生效（例外：动产抵押权合同生效设立；土地承包经营权、地役权合同生效，登记对抗）。
非基于法律行为：合法建造、继承、法院判决/征收决定等，自事实成就/法律文书生效时发生效力，无须公示即可取得，但再处分须先登记。

🔁 交付替代方式

  • 简易交付：受让人已占有（租转买），买卖合同生效时物权变动。
  • 指示交付：转让人请求第三人返还原物的权利代替交付。
  • 占有改定：出让人继续占有，借用约定生效时物权变动（不能用于善意取得交付）。
🚲 【彬哥案例·占有改定】自行车转让

赵某借自行车给钱某，2022.1.10双方达成转让协议。1.11钱某将该车转让给孙某，约定钱某继续使用至2月25日。
解析：1.10钱某通过简易交付取得所有权；1.11钱某与孙某约定占有改定，孙某于约定生效时取得所有权（2021真题改编）。

📋 不动产登记类型

首次登记、变更登记(主体不变内容变)、转移登记(主体变更)、更正登记、异议登记(15日内不起诉失效)、预告登记(90日未申请失效，未经预告权利人同意处分不发生物权效力)。

四、所有权 · 共有（★★★）

国家所有权：矿藏、水流、海域、城市土地等；集体所有权；私人所有权（含法人）。

共有形态（无约定→按份共有，家庭关系除外）

共同共有按份共有
重大修缮/变更全体同意份额≥2/3同意
共有物分割共有的基础丧失或有重大理由随时请求分割（约定除外）
对外债务连带债权债务连带债权债务，内部按份追偿
份额处分-自由转让，其他共有人同等条件优先购买权
💡 优先购买权关键点：仅交易中产生（继承、遗赠无优先权）；行使期间通知载明≥15日，未通知则知道之日起15日或权属转移6个月；两个以上按份共有人协商不成按份额比例行使。

五、善意取得制度（★★★）

构成要件（同时满足）：无权处分 + 受让人善意 + 合理价格 + 动产交付/不动产登记 + 转让合同有效。
❌ 不适用：遗失物、盗窃物（脱手物），占有改定不满足交付要求。

📱 【遗失物不适用善意取得】案例

乙拾得甲的手机，以500元卖给不知情的经销商丙。
解析：乙无权处分，甲未失去所有权；甲可向乙请求损害赔偿，并且自知道受让人丙之日起2年内请求返还原物（但若丙从具有经营资格的经营者购得，甲需支付费用后追偿）。真题正确选项：乙行为无权处分；甲有权请求乙赔偿。

六、用益物权（★★）

口诀：土地建宅居 — 土地承包经营权、建设用地使用权、宅基地使用权、居住权、地役权。
其中建设用地使用权自登记时设立；住宅用地70年，届满自动续期；非住宅届满前1年申请续期。
划拨方式取得的土地使用权转让须经审批；出让转让需完成开发投资总额25%以上。

🆕 集体经营性建设用地出让须经村民会议2/3以上成员同意，且可参照同类国有建设用地执行。

七、担保物权概述（★）

从属性、附条件性、优先受偿性。包括意定担保（抵押权、质权）和法定担保（留置权）。主债权诉讼时效届满，抵押权人丧失胜诉权（抵押人可主张不担责）。

八、抵押权（★★★）

1. 抵押财产范围

可抵押：建筑物、建设用地使用权、海域使用权、生产设备、原材料、正在建造的船舶等。禁止抵押：土地所有权、宅基地、公益设施、权属不明财产、被查封财产。

2. 抵押设立方式

  • 不动产抵押：书面合同 + 登记（登记生效）
  • 动产抵押：合同生效即设立，未经登记不得对抗善意第三人（正常经营买受人规则：无论是否登记，不得对抗已支付合理价款并取得动产的买受人）
🏠 【房地一体+新增建筑物】彬哥案例

2024.1.1甲公司将土地及地上建筑物抵押给乙银行。同年6月新建建筑物，2025年建成。2026年拍卖共得2200万，新建物价值600万。乙银行只能就1600万优先受偿。
知识点：抵押权效力不及于新建物，但处分时须一并处分，价款中新增部分无优先权。

3. 抵押物转让（民法典新规）

抵押人可转让抵押财产，及时通知抵押权人。若约定禁止转让未登记，转让有效且受让人取得无负担所有权（善意）；若禁止转让已登记，则转让不发生物权效力（但受让人代偿除外）。

4. 抵押权顺位 & 超级优先权

登记优先于未登记，均按登记时间先后；未登记按债权比例。动产价款债权抵押权（超级优先权）：标的物交付后10日内办理登记的，该抵押权人优先于除留置权外的其他担保物权人。如：融资租赁出租人、出卖人价款抵押权。

九、质权（★★）

动产质权：书面合同+交付（流质条款无效）。权利质权：有权利凭证的交付设立（汇票、仓单、提单）；基金份额、股权、知识产权、应收账款——办理出质登记设立。
质权人不得擅自使用、转质；负有妥善保管义务。

十、留置权（★★）

法定担保物权，成立要件：合法占有债务人/第三人动产 + 债权已届清偿期 + 同一法律关系（企业之间除外）。留置权人应给债务人60日以上履行期（鲜活易腐除外）。
效力顺位：同一动产上留置权 > 动产超级优先权 > 已登记抵押权/质权(按时间先后) > 未登记抵押权。

🚚 【企业留置例外】案例

运输公司承运甲公司设备，甲公司未支付前次运费，运输公司留置本次设备。个人司机不行，但企业之间留置不受同一法律关系限制（《民法典》448条）。

十一、重点新增点&难点总结（★★★）

  • ✨ 动产抵押超级优先权（价款债权抵押权） — 《民法典》416条，交付后10日内登记，可优先于公示在先的浮动抵押权和其他担保权（留置权除外）。2025大纲考察概率高。
  • ✨ 让与担保 — 形式上转移所有权，实质上担保功能。流质条款无效，但债权人可请求参照担保物权优先受偿（《民法典担保制度解释》68条）。
  • ✨ 正常经营买受人规则 — 动产抵押即使已登记，也不得对抗正常经营活动中的买受人（已付合理价款+取得占有）。
  • ✨ 抵押不破租赁 — 抵押权设立前已出租并转移占有的，原租赁关系不受影响。
  • ✨ 异议登记与预告登记失效期间 — 异议登记15日内不起诉失效；预告登记自能够登记之日起90日内未申请失效。

十二、真正的利他合同与非真正的利他合同（★★★ 合同法+担保物权衔接）新增必考衔接点

在担保、保险以及涉及第三人履行的物权合同中常考。根据《民法典》第522条，分为两种形态：

📌 概念界定

  • 真正的利他合同（不真正利他合同？注意术语纠正）：真正的利他合同 = 第三人直接取得履行请求权。法律规定或者当事人约定第三人可直接请求债务人向其履行债务，第三人未在合理期限内明确拒绝的，第三人可以请求债务人承担违约责任。
  • 非真正的利他合同（即第三人负担/向第三人履行合同）：当事人约定由债务人向第三人履行债务，但第三人未取得独立的履行请求权，仅作为债权受领方，违约时仍由债权人主张违约责任。
对比维度真正的利他合同非真正的利他合同
第三人权利第三人享有直接请求履行的权利，可起诉债务人第三人仅为履行受领人，无独立请求权
拒绝权第三人可在合理期限内明确拒绝无此问题，第三人只是接受履行方
违约责任请求权第三人或债权人皆可主张债务人违约仅债权人可主张违约，第三人无权
典型案例保险合同（受益人直接请求保险金）、信托合同、真正的第三方保理父母为子女付款的买卖合同、指示付款合同
📄 【真正的利他合同·经典】保险合同受益人

甲（投保人）与乙保险公司签订人寿保险合同，指定丙为受益人。丙虽非合同当事人，但依据《保险法》及民法典第522条第2款，丙享有直接向乙保险公司请求支付保险金的权利。若乙拒绝理赔，丙可作为原告起诉，此即真正利他合同。

知识点对应：受益人直接请求权独立于投保人，体现“第三人直接取得履行请求权”。

📄 【非真正的利他合同·指示交付/付款】

甲向乙购买设备，约定乙将设备交付给丙（甲的关联公司），甲向乙支付货款。丙仅为受领指示人，如果乙交付的设备有瑕疵，丙无权直接要求乙承担违约责任，只能由甲主张权利。这属于“向第三人履行”的非真正利他合同。

关联物权变动：指示交付情境（动产转让由第三人直接交付）——第三方占有，通过转让返还请求权完成交付，此时第三人并非合同请求权主体。

🧠 实务难点与考试陷阱

  • 合同约定“向第三人履行”并不当然等于第三人享有独立请求权；必须有“第三人可直接请求”或者法律规定（如《保险法》）。
  • 真正的利他合同成立后，债务人未履行，第三人主张违约责任时，原债权人仍享有债权人的地位（如解除合同权利通常保留）。
  • 在保证合同中，若约定一般保证人向第三方（特定债权人之外的人）履行，则需辨析是否构成真正的利他，通常不构成；而债权人受益的保证合同保证人直接对债权人承担责任不属于利他合同范畴。
  • 与债务加入区分：债务加入是第三人自愿加入履行，第三人成为债务人；利他合同中的第三人是权利人而非债务人。
🔔 法条依据：《民法典》第522条
当事人约定由债务人向第三人履行债务，债务人未向第三人履行债务或者履行债务不符合约定的，应当向债权人承担违约责任（非真正利他）。
法律规定或者当事人约定第三人可以直接请求债务人向其履行债务，第三人未在合理期限内明确拒绝的，第三人可以请求债务人承担违约责任（真正利他）。

📎 与经济法/担保制度关联（综合题命题方向）

1. 物权担保+利他：债务人将抵押物变卖后指示买受人将价款直接支付给担保权人，该约定若明确担保权人可直接请求支付，可能构成真正的利他合同。
2. 让与担保+利他结构：实践中让与担保的收益分配条款涉及第三人时需判断是否赋予独立请求权。
3. 2025年大纲新增“民法典合同编通则解释”背景下，利他合同与代位权、撤销权结合命题可能性上升。

🔍 真题延伸（模拟预测）

甲公司与乙银行订立借款合同，约定由丙公司（关联方）代乙银行收取还款本息，后丙公司未收到款项，丙是否有权起诉甲公司？
分析：若合同中未赋予丙直接请求的权利，则属于“向第三人履行”，丙仅为受领人，无权起诉甲公司；应由乙银行主张违约。如果约定“丙有权直接请求甲公司支付”，则丙享有独立请求权。

无权处分+善意取得公式：无权处分 + 善意 + 合理价格 + 动产交付/不动产登记 = 善意取得（所有权转移）。
不动产物权变动公式：有权处分 + 登记 = 物权变动；动产变动：有权处分 + 交付 = 物权变动（简易交付/指示交付等）。
利他合同：真正的利他 → 第三人独立请求权；非真正的利他 → 仅受领履行。

📝 例题·按份共有与无权处分（彬哥案例）

甲乙按份共有一设备，甲份额1/4，乙3/4。①甲未经乙同意以市价卖给不知情的丙，约定3日后交付 ➜ 设备未交付，丙不能善意取得。②乙将设备卖给丁并交付 ➜ 有权处分，丁取得所有权。无权处分不影响买卖合同效力。
