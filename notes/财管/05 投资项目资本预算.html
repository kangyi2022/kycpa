🔍 基于2026年CPA考纲 & 最新教材（无实质变化，但重难点明确）
⭐ 历年均分≈8分 📌 客观题+主观题高频 🔄 结合本量利/业绩评价

📌 考点一：项目评价方法（★★★）

1. 独立项目评价基本指标

✔ 净现值 (NPV) = Σ未来现金净流量现值 – 原始投资额现值
✔ 现值指数 (PI) = 未来现金净流量现值 ÷ 原始投资额现值 = 1 + NPV / 原始投资额
✔ 内含报酬率 (IRR)：令NPV=0时的折现率 (项目本身报酬率)
  • 决策共性：NPV>0 ↔ PI>1 ↔ IRR>资本成本 → 可行；三者评价独立项目结论一致。
  • IRR计算要点：各年现金流不等时采用逐步测试+内插法；相等时直接内插法。
  • 修订内含报酬率(MIRR)：以资本成本折现现金流出现值和现金流入终值，再计算IRR。
    MIRR = (现金流入终值 / 现金流出现值)^(1/n) - 1 (例题5-4)

2. 辅助方法：回收期法 & 会计报酬率

📆 静态回收期 = M + (第M年尚未回收额) / 第(M+1)年现金净流量
⏳ 动态回收期：考虑现值累计，M + (尚未回收现值)/下年现金流现值
📊 会计报酬率 = 年平均税后经营净利润 / 原始投资额（或平均资本占用）×100%
⚠️ 回收期法缺陷：忽视回收期后收益，动态回收期虽考虑时间价值但仍忽略后期现金流。

3. 互斥项目优选（重点矛盾与解法）

  • ✅ 投资额不同/时间分布不同：以净现值法优先（股东财富最大化）。
  • ✅ 项目期限不同 → 共同年限法 或 等额年金法
    等额年金 = NPV / (P/A, i, n)
    永续净现值 = 等额年金 / 资本成本 （资本成本相同时直接比较等额年金）
  • ‼️ 共同年限法取最小公倍数重置，比较重置后净现值；缺点：重置假设不合理，忽略技术迭代。
🧠 学霸技巧：等额年金法最终比较永续净现值；若资本成本不同必须计算永续净现值，若资本成本相同只需比等额年金。

📌 考点二：新建项目现金流量的估计（★★★）

核心原则：增量现金流量（只有相关成本才计入）

💰 营业期现金流量公式（三法合一）
① 营业现金毛流量 = 税后收入 – 税后付现成本 + 折旧×税率
② = 营业收入 – 付现成本 – 所得税
③ = 税后经营净利润 + 折旧摊销
🏭 建设期 = 长期资产投资 + 垫支营运资本
🏁 终结期 = 收回营运资本 + 资产变现净回收(残值收益纳税/损失抵税)
  • 机会成本：必须考虑因本项目占用资源放弃的其他收益。
  • 沉没成本/非相关成本：如过去调研费、厂房历史购置价（例题5-11）不予考虑。
  • 注意营运资本垫支通常在项目终结点全额回收。
⭐ 常考辨析：利用现有厂房若可出租但有“限制不得出租”则无机会成本；新产品挤占原有销售需考虑竞争关系（增量视角）。

📌 考点三：更新项目现金流量的估计（★★）

1. 决策思路——平均年成本法 (适用于年限不同)

📌 平均年成本 = 现金流出总现值 / (P/A, i, n)
继续使用旧设备初始成本 = 放弃的变现价值 + 变现损失抵税或 - 变现收益纳税
  • 更新决策本质是互斥方案选择：计算现金流出的年均成本，选低者。
  • 旧设备变现价值作为继续使用的初始现金流出（机会成本），同时调整所得税影响。
💡 经济寿命：使固定资产平均年成本最小的使用年限（运行成本上升，持有成本下降的平衡）。

📌 考点四：投资项目折现率的估计（★★）

1. 使用企业当前WACC的条件

  • 项目经营风险与公司当前平均风险相同 + 资本结构不变。

2. 可比公司法（去杠杆/加杠杆）

🔧 步骤① 卸载可比公司财务杠杆：
β资产 = β权益 / [1 + (1-T可比) × (净负债/股东权益)可比]
🔧 步骤② 加载目标公司杠杆：
β权益(新) = β资产 × [1 + (1-T目标) × (净负债/权益)新项目]
📈 股权成本 = Rf + β权益 × 市场风险溢价
WACC = 税后债务成本×负债比重 + 股权成本×权益比重
典型真题(例题5-14)：需要先卸载可比公司β，再加载目标公司资本结构，得到项目β权益。2025年多选题曾考。

📌 考点五：投资项目的敏感分析（★★）

  • 最大最小法：令NPV=0，求某一变量的临界值（同向变量最小值，反向变量最大值）。
  • 敏感程度法：敏感系数 = 目标变量变动百分比 / 选定因素变动百分比；|敏感系数|>1为敏感因素。
  • 局限性：单变量变动假设，未考虑变量关联性及发生概率。
📌 与NPV同向变动的因素（如销量、单价）临界值是最小值；反向因素（如初始投资、变动成本）临界值是最大值。

🎯 本章重难点归纳

✅ 重点：独立项目评价指标（NPV、IRR、PI）的决策一致性；互斥项目年限不同时的共同年限法/等额年金法；新建项目现金流构成（机会成本、营运资本、折旧抵税）；可比公司法估计项目资本成本。
⚠️ 难点：互斥项目出现矛盾时的处理（投资额/期限不同导致），尤其共同年限法重置净现值的计算；存在所得税情况下固定资产更新决策中变现损失抵税、折旧抵税时点的处理；加载/卸载财务杠杆时所得税率调整。敏感分析计算临界值。
